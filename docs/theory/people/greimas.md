# Algirdas Julien Greimas

Lithuanian-French semiotician (1917–1992). Extended Propp into a full narrative grammar.

## Key ideas
- **Actantial model** — six actants: Subject, Object, Sender, Receiver, Helper, Opponent
- **Narrative grammar** — surface narrative structures derived from deep logical structures
- **Semiotic square** — four-term logical structure mapping oppositions and contradictions
- **Isotopy** — semantic coherence within a text; repeated semic categories

## Relevance
The actantial model generalizes Propp beyond folktales. The semiotic square is a useful tool for mapping thematic oppositions (life/death, freedom/constraint) as graph nodes. Greimas provides the most developed formal grammar for narrative — closest to what Plotz aims to operationalize.

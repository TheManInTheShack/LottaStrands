# Charles Sanders Peirce

American philosopher and logician (1839–1914). Founder of pragmatism; developed a triadic theory of signs.

## Key ideas
- **Icon / Index / Symbol** — three modes of sign relation to object
  - *Icon*: resemblance (a portrait, a map)
  - *Index*: causal/contiguous link (smoke → fire, a pointing finger)
  - *Symbol*: arbitrary convention (words, most language)
- **Semiosis** — the process by which a sign produces an interpretant
- **Triadic model**: Sign → Object → Interpretant (not dyadic like Saussure)

## Relevance
The icon/index/symbol distinction is useful for classifying narrative elements — flashbacks are indexical, metaphors are iconic, proper nouns are symbolic.

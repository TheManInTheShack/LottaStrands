# Vladimir Propp

Russian folklorist (1895–1970). Morphologist of the folktale.

## Key ideas
- **31 narrative functions** — the irreducible structural units of the Russian folktale
- **7 character types (actants)** — Hero, Villain, Donor, Helper, Princess, Dispatcher, False Hero
- Functions occur in fixed sequence; not all need be present but order is preserved
- First systematic attempt to describe narrative grammar

## Relevance
Propp's functions are a candidate ontology for narrative events in Plotz. The actant types are a candidate for character classification. *Morphology of the Folktale* (1928) is the direct ancestor of structuralist narratology.
